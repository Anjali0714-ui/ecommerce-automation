Ecommerce Automation Framework
🔹 Project Description
This project is an automation testing framework for an ecommerce website using Selenium WebDriver.

🔹 Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
🔹 Features
- Login Test
- Product Selection
- Add to Cart
- Checkout Flow
🔹 Project Structure
- src → Test scripts & page classes
- test-output → Test reports
- pom.xml → Dependencies
🔹 How to Run
1. Clone the repo
2. Import as Maven project in Eclipse
3. Run TestNG file
